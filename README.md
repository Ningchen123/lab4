# Ecommerce Platform

## Description
Ecommerce Platform - это полнофункциональная система интернет-магазина, разработанная для управления онлайн-продажами. Система включает модули управления пользователями, товарами, заказами и поставщиками. Поддерживает ролевую модель доступа (покупатель, менеджер, администратор) и предоставляет REST API для интеграции с веб- и мобильными приложениями.
